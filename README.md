## 📁 프로젝트 개요

### 📂 화면 구성 & 기능

- 메인 화면
- 로그인
  - 회원가입, 로그아웃
- 마이페이지
  - 회원 정보 수정, 회원 탈퇴
- 둘러보기(카테고리별로 정보 출력)
  - 전체, 박물관, 전시회, 미술관, 팝업스토어
- 지도
  - 길찾기
  - 찜하기
  - 더보기
    - 방문자 리뷰 등록
- 찜 리스트
  - (지도 -> 찜하기) 한 목록 리스트를 출력
