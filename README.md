# 🏛 MyPlace

**내 위치 근처 전시회/팝업 스토어 찾는 웹사이트 프로젝트 (2023/11/07 ~ 2023/11/23)**

## 기획 의도

서울 시민의 문화시설에 대한 정보를 제공하고자 제작
근처에 있는 미술관, 박물관, 팝업스토어, 전시회를 한 번에 빠르게 찾기 위해 기획
